# 💰 Projeto Fintech - FIMP

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte de uma atividade avaliativa da faculdade FIAP, com o objetivo de aplicar conhecimentos em **HTML**, **CSS** e **Tailwind CSS** na construção de uma interface de uma aplicação do tipo **Fintech**.

A proposta consiste em criar uma tela responsiva relacionada a um sistema financeiro digital. Neste caso, foi desenvolvida uma **tela de login**, com identidade visual própria e foco na experiência do usuário.

---

## 🎯 Objetivos

- Criar uma interface moderna utilizando HTML e CSS
- Aplicar o framework **Tailwind CSS** para estilização
- Garantir **responsividade** para diferentes dispositivos
- Manter a separação entre estrutura (HTML) e estilo (CSS)
- Publicar o projeto no GitHub

---

## 🖥️ Tecnologias Utilizadas

- HTML5
- CSS3
- Tailwind CSS

---

## 📱 Funcionalidades da Tela

- Tela de login com:
  - Campo de e-mail
  - Campo de senha
  - Botão de login
- Links auxiliares:
  - "Esqueceu a senha?"
  - "Criar conta"
- Layout responsivo (adaptado para mobile e desktop)
- Identidade visual com uso de cores, tipografia e imagens personalizadas

## 🚀 Como Executar o Projeto

Este projeto não requer instalação de dependências ou servidor.

### ✅ Passo a passo:

1. Faça o download ou clone o repositório:

```
git clone git@github.com:rocha-henrique/FIMP.git
```

2. Acesse a pasta do projeto:

cd seu-repositorio

3. Abra o arquivo `index.html` diretamente no navegador:

- Clique duas vezes no arquivo  
**ou**  
- Clique com o botão direito → "Abrir com" → Navegador
